## BitNet

> 1位大语言模型官方推理框架

**作者:** [microsoft](https://github.com/microsoft.png) | **仓库:** [Github 链接](https://github.com/microsoft/BitNet)

![Language](https://img.shields.io/badge/C++-f34b7d.svg?style=flat-square&logo=C++&logoColor=white) ![Stars](https://img.shields.io/badge/Stars-14819-yellow.svg?style=flat-square) ![Forks](https://img.shields.io/badge/Forks-1043-blue.svg?style=flat-square) ![Recent Stars](https://img.shields.io/badge/Recent%20Stars-627-brightgreen.svg?style=flat-square)

## 项目简介

**微软开源BitNet项目摘要**

BitNet是微软专为1-bit大语言模型（如BitNet b1.58）推出的高效推理框架，核心特点包括：

- **高性能**：ARM/x86 CPU推理速度提升1.37x-6.17x，能耗降低55%-82%，单CPU可支持100B参数模型（5-7 token/秒）。
- **技术资源**：提供优化内核、演示视频、技术报告及MIT许可代码，获14.8k星标。

**关键进展**

- 发布2B/3B参数模型（Hugging Face）、4位激活研究及CPU高效框架bitnet.cpp。
- 基于llama.cpp整合T-MAC技术，支持x86/ARM架构，覆盖0.7B-10B参数模型。

**使用指南**

- 要求Python≥3.9/CMake≥3.22，支持快速部署（克隆→下载模型→运行推理脚本）。
- 提供参数化工具（如线程控制、温度调节）及基准测试脚本。

**目标**：推动1-bit模型在边缘计算和大规模训练中的应用。

## fastmcp

> 🚀 快速、Pythonic 方式构建 MCP 服务器与客户端

**作者:** [jlowin](https://github.com/jlowin.png) | **仓库:** [Github 链接](https://github.com/jlowin/fastmcp)

![Language](https://img.shields.io/badge/Python-3572A5.svg?style=flat-square&logo=Python&logoColor=white) ![Stars](https://img.shields.io/badge/Stars-6377-yellow.svg?style=flat-square) ![Forks](https://img.shields.io/badge/Forks-331-blue.svg?style=flat-square) ![Recent Stars](https://img.shields.io/badge/Recent%20Stars-341-brightgreen.svg?style=flat-square)

## 项目简介

**FastMCP 项目摘要**

FastMCP 是一个基于 Python 的高效工具库，用于快速构建符合 MCP（Model Context Protocol）协议的服务器和客户端，专为 LLM 应用设计。

**核心特点**

- **简化开发**：通过装饰器（如 `@mcp.tool`）定义工具、资源和提示模板，自动处理协议细节，支持异步和类型提示（Pydantic）。
- **模块化功能**：支持子应用挂载、服务组合、代理服务器及 OpenAPI/FastAPI 集成，V2 新增客户端 LLM 采样和 OpenAPI 生成能力。
- **安全协作**：提供文件访问控制（RootsList）、上下文管理及多传输协议（SSE/Stdio/WebSocket），确保服务端与客户端灵活安全交互。

**项目状态**

- 开源（Apache-2.0），GitHub 6.4k 星标，支持 PyPI 安装（`uv pip install fastmcp`）。
- 官网 [gofastmcp.com](https://gofastmcp.com) 提供完整文档和示例（如工具调用、图像处理、LLM 集成等）。

**协议基础**  
MCP 标准化了 LLM 与服务的交互，包含：

- **资源**（类 GET 端点）、**工具**（类 POST 端点）、**提示模板**（可复用交互模式）。

**适用场景**  
快速开发 Claude 等平台的集成服务，支持复杂业务逻辑（如数据库操作、API 代理）与模块化部署。
